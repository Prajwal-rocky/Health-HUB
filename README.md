# Health-HUB
HealthHub is a smart healthcare platform designed to help users monitor their health, track wearable data, and receive personalized wellness guidance. The platform focuses on preventive care by combining health tracking, intelligent insights, and simple user-friendly tools.  HealthHub allows users to manage their health in one place. 

Key Features

Health Monitoring
Track important health metrics such as heart rate, oxygen level (SpO₂), steps, sleep data, and activity levels through wearable device integration.

Wearable Integration
Connect wearable devices via Google Fit or Fitbit to automatically sync health data to the dashboard.

Disease Selection
Users can select diseases or conditions to receive personalized health insights and recommendations.

Diet Planning
Indian-style veg and non-veg meal plans tailored to user health conditions and BMI.

AI Health Assistant
Provides general health tips, lifestyle guidance, and disease awareness.

Water Reminder
Helps users maintain hydration with hourly reminders and daily progress tracking.

Demo Payment System
Supports subscription simulation using Indian Rupees (₹) for testing premium features.

Admin Dashboard
Admin access allows monitoring of users, feedback, subscriptions, and health insights.
